# Chore Management

Chore delegatioon website that offer families and households to distribute tasks and chores evenly. This provide a
practical real-life app that is fun and challenging.

_Technology:_
 - Node.js
 - Angular.js
 - Bootstrap
 - Express
 - Socket.io
 - MongoDB

_Features:_
 - Drag/Drop
 - Login/Registration Validation
